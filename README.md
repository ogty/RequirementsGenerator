### `config.json`

`static/config.json`ファイルの`ignores`に非表示にしたいディレクトリ名を追記すると、
そのディレクトリ名を含むパスが表示されなくなります。
また、パスを格納している`tree.json`の容量が小さくなります。

***

### 注意

デスクトップに大量のフォルダがある場合は正常に動作しない場合があります。
